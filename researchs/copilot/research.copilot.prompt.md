# IoT-Security-Research Prompt

Bu araştırmanın amacı, GitHub Copilot gibi yapay zeka destekli araçların IoT güvenlik araştırmalarında nasıl kullanılabileceğini incelemektir. Özellikle aşağıdaki sorulara odaklanılacaktır:

1. GitHub Copilot, IoT güvenlik açıklarını tespit etmede nasıl yardımcı olabilir?
2. Kod önerileri, yapılandırma dosyaları ve güvenlik politikaları açısından Copilot’un katkısı nedir?
3. Açık kaynak IoT güvenlik projeleriyle entegrasyon potansiyeli nedir?
4. Copilot’un önerdiği kodların güvenlik açısından değerlendirilmesi nasıl yapılmalıdır?

Araştırma, hem teknik analiz hem de pratik uygulama örnekleriyle desteklenecektir.
